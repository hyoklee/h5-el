# h5-el
Emacs HDF5 mode

## Usage
```
M-x h5-mode
```

It's similar to Dired.

### View Everything (Datasets, Attributes, and Group)
#### Collapse/Expand Group
`RET` open-group

`+` create-group

`d` delete-group

`s` sort-group


### View Dataset as an Image
`i` view-as-image

`e` export-as-image

### View Dataset as Text
`RET` view-as-text

`E` export-as-csv

### View Attribute
`RET` view-attribute

### Search Value
`Ctrl-s` search

### Quit
`q` quit-window
