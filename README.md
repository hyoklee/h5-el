# h5-el
Emacs HDF5 mode

## Usage
```
M-x h5-mode
```
### View Evertying (Datasets, Attributes, and Group)
#### Collapse/Expand Group

`RET` open-group

`+` create-group

### View Dataset as an Image
`i` view-as-image
### View Dataset as Text
`RET` view-as-text
### View Attribute
`RET` view-attribute
### Quit
`q` quit-window
